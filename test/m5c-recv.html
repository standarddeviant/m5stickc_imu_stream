This is a page to receive IMU data from an M5Stick-C device. The M5Stick-C device contains an ESP32 that streams IMU data with a WebSockets. The ESP32 inside the M5Stick-C maintains a WebSocket server that supports multiple WebSocket clients - like this web page. For efficiency, the data is packaged up with msgpack, which is like a binary version of JSON.

This example is offered as-is for demonstration purposes only.
